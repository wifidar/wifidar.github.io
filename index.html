Wifidar Landing Page

Welcome to the WiFidar project landing page.

This page contains links to the source code used in the WiFidar project. This is the software portion a 2015 Syracuse University senior design project with the goal to build a radar system that operates at WiFi frequencies.

This page should be considered a work in progress, as more information may be added later.

Links

The source code for the software used in the project can be found in the repositories below:

WiFidar_FPGA - contains the code used on the FPGA to sample the IF waveform.

WiFidar_RTI - contains the code used on the computer to pull in samples from the FPGA and display them in an RTI plot.
